Nixvim Samples
=================================================

This folder contains the samples that can be used to verify the nixvim solution will work with the expected language
or configuration.

Available sample projects:

- `cplusplus/` placeholder for future C++ sample work
- `dlang/` Dub-based D sample
- `elixir/` Mix-based Elixir sample
- `erlang/` Rebar3-style Erlang sample
- `golang/` Go module sample
- `haskell/` simple Haskell sample
- `javascript/` Node.js package sample
- `kotlin/` Gradle multi-module Kotlin sample
- `python/poetry-demo/` Poetry-based Python sample
- `rust/` Cargo-based Rust sample
- `typescript/` TypeScript package sample
- `zig/` Zig build sample

The repository root `.nvim/dap.lua` contains sample DAP configurations for the projects in this directory.
